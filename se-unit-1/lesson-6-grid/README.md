# Lesson 6 Notes: CSS Grid

## Vocabulary
* grid container
* grid items
* grid lines
* grid tracks
* `fr` or fraction unit
* explicit grid vs. implicit grid
* `minmax()`

## Key Grid Properties
### Grid Container Properties
* `display: grid`
* `grid-template-columns`
* `grid-template-rows`
* `grid-template-areas`
* `grid-auto-rows`
* `grid-auto-columns`
* `grid-column-gap`
* `grid-row-gap`
* `minmax()`

### Grid Item Properties
* `grid-column-start`
* `grid-column-end`
* `grid-row-start`
* `grid-row-end`
* `grid-area`

## Essential Questions
* How do CSS Flexbox and Grid differ?
* How do you span grid items across multiple rows/columns?
* How can we use fractional units to create fluid layouts?
* What are named grid areas and how do they assist us in laying out a page?
